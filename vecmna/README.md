推荐开发环境
------------------

- pycharm
- macOS
- python3.5+

注意事项
-----------

* 我们不对python3.5以下版本及windows提供支持


知识储备要求[自行学习]
---------------------

- 了解 `pytest` 的基本用法
- 了解 `coverage` 的基本用法



快速预览
----------

```
pip install -r requirements.txt
pip install -r requirements.test.txt
pytest
```
