Flask-Kickstart
=================

推荐工作环境
------------------

- pycharm
- macOS
- python3.5+
- docker


注意事项
-----------

* 我们不对python3.5以下版本提供支持
* 项目保留名称： **lego** 、 **hive** 、
  请不要将项目命名为此名称，后续我们可能会开源相关的组件，以减少冲突。


快速开始
----------


快速预览
^^^^^^^^^^^^
需要占用5000端口

.. code-block::

    docker-compose up -d
    curl http://127.0.0.1:5000
